# rk_app
**rck_pp** - Web service by ruby. Create simple and lightweight app 
Fot

## Dependencies:
```sh
ruby '2.7.2'

gem 'rack'
gem 'rack-parser'
gem 'rake'
gem 'require_all'
gem 'activesupport'
gem 'config'
gem 'dry-system'
gem 'dry-transaction'
gem 'dry-view'
gem 'rubocop'
gem 'rubocop-rake'
gem minitest'
```
## Install:
```sh
git clone
```
## How is it works?:
```s
bundle exec rackup

Frist start: bundle exec rackup
After start you should try make a ping action
      http://localhost:9292/ping.json
If a response is "pong", server is successfully runing     
```
